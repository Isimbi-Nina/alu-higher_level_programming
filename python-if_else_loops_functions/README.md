# Python - if/else, loops, functions

## Description
This project focuses on fundamental Python concepts such as conditionals, loops, and functions. It is part of the ALU Higher Level Programming curriculum.

You will learn how to control program flow using if, elif, and else statements, how to iterate using for and while loops, and how to create and use functions in Python.

## Learning Objectives

By the end of this project, you should be able to explain:

- Why Python programming is powerful and easy to use
- Why indentation is very important in Python
- How to use if, if...else, and elif statements
- How to write and use comments in Python
- How to assign values to variables
- How to use while and for loops
- The difference between Python for loops and C for loops
- How to use break and continue
- How to use else with loops
- What the pass statement does
- How to use range()
- What functions are and how to define them
- What return values are in functions
- Variable scope in Python
- What a traceback is
- Basic arithmetic operators in Python

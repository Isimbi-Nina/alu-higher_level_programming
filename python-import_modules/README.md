# python-import_modules

This project covers the basics of Python modules, imports, and command
line arguments.

## Learning Objectives

- Why Python programming is awesome
- How to import functions from another file
- How to use imported functions
- How to create a module
- How to use the built-in function `dir()`
- How to prevent code in your script from being executed when imported
- How to use command line arguments with your Python programs

## Tasks

| File | Description |
| --- | --- |
| `add_0.py` | Module containing an `add` function |
| `0-add.py` | Imports `add` from `add_0` and prints `1 + 2 = 3` |
| `calculator_1.py` | Module containing `add`, `sub`, `mul`, `div` functions |
| `1-calculation.py` | Imports and calls all four calculator functions |
| `2-args.py` | Prints the number and list of arguments passed to it |
| `3-infinite_add.py` | Prints the sum of all arguments passed to it |
| `4-hidden_discovery.py` | Prints public names defined in `hidden_4.pyc` |
| `variable_load_5.py` | Module containing a variable `a` |
| `5-variable_load.py` | Imports `a` from `variable_load_5` and prints it |

## Requirements

- Ubuntu 20.04 LTS, python3 (3.8.5)
- All files start with `#!/usr/bin/python3`
- All files end with a newline
- All files are executable
- Code follows pycodestyle (2.7.*)
